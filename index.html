About This Project

Overview

T&J Heritage is a modern single-page fashion catalog application inspired by premium retail brands. This project demonstrates dynamic client-side rendering, interactive catalog browsing, and a fully functional shopping cart — all built without external frameworks.

Meet the Team

Zaif Zunayed Developer
Tyler Mulvey Developer

Course Information

Course: COMP 3512 – Web Development II

Assignment: Product Catalog SPA

Institution: Mount Royal University

Key Features

  • Responsive single-page architecture with modular JavaScript
  • Dynamic filtering, sorting, and live search capabilities
  • Interactive product detail pages with size and color selection
  • "Quick Add" modal for streamlined purchasing experience
  • LocalStorage-powered shopping cart with real-time quantity updates
  • Premium Zara-inspired clean UI with smooth UX patterns
  • Mobile-first responsive design across all devices

Technologies Used

JavaScript ES6+ HTML5 CSS3 LocalStorage API Responsive Design SPA Architecture

Development Philosophy

This project embodies clean architecture principles and reusable view components while maintaining the sophisticated aesthetic of premium retail websites. Every page transition, interaction, and visual element is meticulously designed to deliver a modern, fast, and user-centered experience that rivals production e-commerce platforms.

Project Highlights

  • Zero external framework dependencies - pure vanilla JavaScript
  • Modular component-based architecture for maintainability
  • Performance-optimized with efficient DOM manipulation
  • Accessibility-focused with semantic HTML and ARIA labels
  • Cross-browser compatible and tested across multiple devices
